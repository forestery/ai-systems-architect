## 为什么写这本书

2026 年初，作者在研究多 Agent 协同编程时发现：**中文世界对这个话题的系统性内容几乎为零。**

Anthropic 的报告、Yegge 的实验、Cursor 的架构演进——这些信息散落在英文技术博客、GitHub README 和行业报告中。中国工程师想了解这个领域，需要跨越语言、渠道和语境三重障碍。

这本书试图填补这个空白：**给中文世界的软件工程师一份诚实的、深入的、可操作的地图。**
